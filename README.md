## ⭐주요 기능 



###  경기도 데이터드림 공공 API 활용
- 식당 이름 / 주소 / 좌표값 / 등등을 활용하여 내용 출력
- 경기도 내 안심식당 목록 사용

### 카카오 맵 API 활용
- 식당 지도 정보 출력
- 식당 상세위치 로드뷰 출력
  
### jQuery mobile을 활용한 화면 구현
